📱 Gerador de QR Code Personalizado
📝 Descrição do Projeto

Este projeto consiste em uma interface web para geração de QR Codes personalizados, permitindo que os usuários criem códigos QR estilizados a partir de links, textos ou outros dados.

A aplicação oferece diversas opções de personalização, como alteração de tamanho, margens, cores, formato dos pontos, estilo dos cantos e inclusão de imagens no centro do QR Code, tornando o código visualmente mais atrativo e adequado para diferentes aplicações como marketing, cartões digitais e compartilhamento de links.

O sistema utiliza uma biblioteca JavaScript especializada em geração de QR Codes estilizados, permitindo que o usuário visualize o resultado em tempo real e faça o download do QR Code gerado em diferentes formatos.

⚙️ Funcionalidades do Sistema

O sistema permite ao usuário configurar diferentes parâmetros para gerar QR Codes personalizados:

Inserção de URL ou texto para geração do QR Code
Upload de imagem para inserção no centro do código
Definição de largura e altura do QR Code
Ajuste de margem do código
Personalização de estilo dos pontos (Dots)
Alteração do formato dos cantos
Configuração de cores e fundo
Exportação do QR Code gerado
Download da imagem em formato PNG
📊 Resultados e Aprendizados

Durante o desenvolvimento e uso da ferramenta, foi possível observar importantes aspectos relacionados à geração de QR Codes personalizados:

Personalização visual: O design pode ser adaptado para identidade visual de marcas.
Interatividade em tempo real: As alterações feitas pelo usuário são refletidas imediatamente na visualização do QR Code.
Usabilidade: A interface simples permite gerar QR Codes rapidamente sem necessidade de conhecimento técnico.

Além disso, o projeto demonstra como bibliotecas JavaScript podem ser utilizadas para criar ferramentas visuais interativas diretamente no navegador.

🔧 Como Executar
Abra o projeto no navegador ou ambiente web.
Insira o link ou texto que deseja transformar em QR Code.
Configure as opções de personalização (tamanho, cor, estilo).
Visualize o QR Code gerado em tempo real.
Clique em Download para salvar o QR Code no computador.
📌 Conclusão

O projeto demonstra como é possível criar um gerador de QR Codes estilizados utilizando tecnologias web simples, oferecendo flexibilidade de personalização e facilidade de uso.

Ferramentas desse tipo são úteis para marketing digital, compartilhamento de informações e criação de materiais visuais interativos, tornando o QR Code não apenas funcional, mas também visualmente atrativo.
